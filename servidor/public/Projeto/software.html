Sobre o software

Este software foi desenvolvido para facilitar a avaliação e o gerenciamento de patentes de forma simples e eficiente. Com uma interface intuitiva, ele permite cadastrar, listar e avaliar patentes, garantindo que apenas as pendentes apareçam para análise. Após a avaliação, as patentes são automaticamente movidas para a lista de concluídas, promovendo organização e controle no processo.
